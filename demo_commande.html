Démo - Mode commande

Commande de deux leds

Matériel nécessaire

  • Deux résistances de 220 ohms
  • Une led rouge
  • Une led verte
  • Une carte Arduino équipée d'une breadboard
  • Quelques fils

Montage

Sur une carte Arduino, réaliser le montage suivant:

Script Arduino

  • Télécharger le script leds.ino en cliquant sur le lien
  • Téléverser ce script dans la carte à l'aide de l'IDE Arduino

Utilisation

  • Cliquer sur le bouton Connexion ci-dessous et sélectionner le port correspondant à la carte.
  • Le bouton Rouge allume la led rouge
  • Le bouton Vert allume la led verte
  • Le bouton Stop éteint les deux leds

Code Javascript

Le code Javascript nécessaire à cette application est le suivant:

mode = "commande";
commandes = [{texte_bouton: "Rouge", arduino: "rouge"},
             {texte_bouton: "Vert", arduino: "vert"},
             {texte_bouton: "Stop", arduino: "stop"}];
